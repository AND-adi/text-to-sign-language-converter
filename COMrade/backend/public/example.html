Welcome to the Demo

This is a sample webpage with important content that demonstrates how the COMRADE accessibility widget works. The widget adds an overlay to any website, allowing users to customize their experience based on their accessibility needs.

Key Features: The widget provides multiple accessibility profiles including dyslexia support, ADHD focus mode, low vision support, and motor impairment accommodations.

Try Different Profiles

Click the accessibility button in the bottom-right corner to open the widget panel. You'll see several profiles to choose from:

Standard UI

The default interface without any modifications.

Dyslexia Support

Enhanced spacing and readability features for easier reading.

ADHD Focus

Highlights key terms to help maintain attention and focus.

Low Vision

Larger text and high contrast for better visibility.

Motor Impairment

Larger click targets and reduced motion for easier interaction.

Integration Instructions

To add the COMRADE widget to your own website, simply include this script tag in your HTML (replace YOUR_API_TOKEN with your actual token): 

<script src="http://localhost:3000/comrade-widget.js" 
        data-comrade-token="YOUR_API_TOKEN">
</script>
Getting a Token: Visit the Admin Dashboard to generate your API token. You'll need the admin key to access it.

About COMRADE

COMRADE (Adaptive Interface Designer) is a comprehensive accessibility solution that makes websites more inclusive for everyone. By providing an easy-to-integrate widget, website owners can offer powerful accessibility features without extensive development work.

The system uses a secure API token approach, allowing centralized management of accessibility features across multiple websites while maintaining user privacy and preferences.
